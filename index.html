Typed Manifest

Provides easy access to META-INF/MANIFEST.MF on the classpath.

Usage

See the samples module.

Back Story

I was building an executable uberjar and correctly populating the Implementation-Version of META-INF/MANIFEST.MF. I wanted the manifest to be the sole source of truth for that information, but I also wanted to print it when invoking java -jar myjar.jar --version. This code isn't unique to my application, so here it is as a reusable jar.
